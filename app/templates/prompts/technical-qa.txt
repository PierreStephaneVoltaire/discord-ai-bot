**Question: "What's the difference between map and filter?"**
Good: "map transforms each element in an array and returns a new array of the same length. filter tests each element and returns a new array containing only elements that pass the test. 

Examples:
- map: `[1,2,3].map(x => x*2)` → `[2,4,6]`
- filter: `[1,2,3].filter(x => x>1)` → `[2,3]`"
