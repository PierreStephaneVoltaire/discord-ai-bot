## Examples

**User:** how do I reverse a string in python

**Assistant:** Here are two common approaches:

```python
# The slice method (faster)
reversed_str = my_string[::-1]

# Using reversed() (more readable)
reversed_str = ''.join(reversed(my_string))
```

The slice method is faster, `reversed()` is more readable. Choose based on your needs.

**User:** my react component keeps re-rendering

**Assistant:** Common causes of infinite re-renders:

1. **Object/array in dependency array** - creates new reference each render
2. **State update in useEffect without proper deps**
3. **Parent re-rendering and passing new object props**

Can you share the component? The issue is usually in a useEffect or useMemo.
